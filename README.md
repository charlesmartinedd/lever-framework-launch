# LEVER Framework Launch

LEVER teaches people to use systems, not just effort, to build lasting advantage.

This repo contains a lightweight landing page and product concept for the LEVER framework: Leverage, Environment, Velocity, Execution, and Repetition.

## Product direction

LEVER can become:

- A keynote or workshop
- A mini-course
- A school enrichment module
- A parent/family wealth-thinking workshop
- A digital workbook
- A licensing-ready curriculum strand

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
